hydrothings.components.sensors.utils.metadata_validator(value, values)

Validation for sensor metadata field.

Runs a check on the sensor metadata field comparing it to the encoding_type field if present. This check ensures that the metadata value is consistent with the encoding type.

Parameters:

  • value (Any) – The metadata value passed to the request.

  • values (dict) – The set of values included in the body of the request.

Returns:

The output value.

Return type:

Any
